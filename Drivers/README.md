## About These Drivers

The driver packages in this directory were extracted directly from the official Intel Rapid Storage Technology (RST) installer, available at:

<https://www.intel.com/content/www/us/en/download/19512/intel-rapid-storage-technology-driver-installation-software-with-intel-optane-memory-10th-and-11th-gen-platforms.html>

Extraction was performed using the following command:

```
SetupRST.exe -extractdrivers
```

These drivers are provided here exactly as extracted, without any modification. All files and associated intellectual property are the sole property of Intel Corporation.
